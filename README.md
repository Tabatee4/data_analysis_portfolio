# Data Analyst Portfolio

This repository showcases my data analysis projects — from data cleaning and SQL querying to business reporting and Power BI visualization.
Each project highlights different technical and analytical skills used in real-world analytical workflows.

Projects
1. Restaurant Data Cleaning (Python)

Cleaned and validated a raw CSV dataset of restaurant transactions.

Applied data validation, logical consistency checks, and missing value handling.

Result: a clean dataset ready for sales analysis and visualization.
Skills: Python, Pandas, Data Cleaning

2. Hospital Data Analysis (Excel)

Conducted exploratory and descriptive analysis of hospital data.

Built pivot tables, charts, and interactive dashboards.

Result: insights into patient demographics, satisfaction, and occupancy.
Skills: Excel, Data Visualization, Reporting

3. Netflix Data Analysis & Visualization (SQL, Python, Power BI)

Connected to PostgreSQL database and analyzed viewing patterns, revenue trends, and user activity.

Exported data to Power BI to create an interactive two-page report (“Netflix Report”).

Result: dynamic visualization of key metrics across users, content, and revenue.
Skills: SQL, Python, Power BI, Data Integration

Tools and Skills

Languages: Python, SQL

Libraries: Pandas, SQLAlchemy, Matplotlib

Visualization: Power BI, Excel

Core Skills: Data Cleaning, Query Design, Data Modeling, Storytelling with Data

Purpose

This portfolio demonstrates my practical data analysis skills and ability to deliver complete analytical workflows — from data extraction and transformation to visualization and insight communication.
